# Diceware
Diceware passphrase generator

Diceware is a technique that uses dice to produce random text for passphrases and other uses. The Diceware method provides an easy way to create strong passphrase that are easy to remember.

Recommended length for passphrase is six words. 
Six words have 77.5 bits of entropy.

The word list consists of 7776 words.
There are 7776^n combinations (where n is the number of words). 

Refer to Diceware's FAQ: http://world.std.com/~reinhold/dicewarefaq.html
